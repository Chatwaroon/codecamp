พัคโบกอม หรือว่า ปาร์ค โบกอม ดาราหนุ่มหน้าใสที่แฟนซีรี่ย์เกาหลี

พ่อหนุ่มรามยอน

    ประวัติพัคโบกอม

  • เกิด:16 มิถุนายน ค.ศ. 1993 (27 ปี)
  • การศึกษา:มหาวิทยาลัยมย็องจี(เอกภาพยนตร์และการดนตรี)
  • อาชีพนักแสดง
  • ปีปฏิบัติงาน 2011–ปัจจุบัน
  • ตัวแทน บลอสซัมเอนเตอร์เทนเมนต์
  • ชื่อเกาหลี ฮันกึล 박보검
  • มีผลงานการแสดงครั้งแรกในภาพยนตร์ในปี ค.ศ. 2011 เรื่อง พยานมืดปมมรณะ
  • ได้รับเสียงวิจารณ์ที่ดีกับความหลากหลายบทบาทของเขาในละครโทรทัศน์หลาย ๆ เรื่อง เช่น อี ยุน-ฮู ในละครเรื่อง จังหวะรัก หัวใจดนตรี (2014)
  • ค.ศ. 2016 เขาเป็นพิธีกรในรายการ มิวสิกแบงก์ ร่วมกับแพ จู-ฮย็อนทางช่องเคบีเอส 2
