อยากเก่งคอมพิวเตอร์ ต้องทำไง

                      ทักษะคอมพิวเตอร์ กลายเป็นสิ่งสำคัญที่ไม่อาจปฏิเสธได้แล้วสำหรับการทำงานในสมัยปัจจุบัน
            ความสามารถในการใช้คอมพิวเตอร์เป็นสิ่งที่จำเป็นอย่างมากในโลกแห่งเทคโนโลยี 4.0
            ไม่ว่าจะเป็นคอมพิวเตอร์หรือการใช้มือถือล้วนมีต้องการทักษะเพื่อช่วยในการทำงานมากยิ่งขึ้นหากคุณสามารถ
            ใช้โปรแกรมคอมพิวเตอร์ต่างๆได้อย่างคล่องแคล่ว
            เชื่อได้ว่ามีผลในการได้งานเป็นอย่างมากโดยเฉพาะผู้ที่ไม่มีประสบการณ์ทำงานหรือเด็กจบใหม่
                      โปรแกรมพื้นฐานที่คุณควรจะศึกษาและใช้งานให้ได้อย่างคล่องแคล่ว ไม่ว่าจะเป็นโปรแกรม Microsoft Word
            สำหรับพิมพ์งาน, Microsoft Excel ในการใส่สูตรต่างๆเพื่อคำนวณตัวเลข และการนำเสนอแผนงานต่างๆ
            ด้วยโปรแกรม Microsoft PowerPoint(ซึ่งหลายๆคนตอนสัมภาษณ์งาน อาจต้องนำเสนอผลงานกับผู้บริหารหรือ
            หัวหน้าฝ่ายที่เกี่ยวข้องมักเลือกใช้โปรแกรมดังกล่าว)

           
  1. คุณอยากพัฒนาทักษะคอมพิวเตอร์ ให้เชี่ยวชาญในด้านใด ? 

                        หลายคนที่ยังเริ่มต้นไม่ถูก คุณลองถามตัวเองก่อนว่า ต้องการพัฒนาทักษะคอมพิวเตอร์
            ในด้านใดเป็นพิเศษ อาจรวมถึงความชอบและการศึกษาที่เรียนมาด้วย ตั้งเป้าหมายว่าจะเริ่มต้นตั้งแต่วันไหน 
            จดบันทึกลงไป  จากนั้นให้ประเมินว่าคุณจะใช้เวลาเท่าไรในการฝึกฝนทักษะนั้นจนสามารถทำสำเร็จได้ตามเป้า
             เช่น คุณเป็นคนที่พิมพ์งานช้ามาก เนื่องจากยังพิมพ์สัมผัสไม่คล่อง ระหว่างพิมพ์ยังต้องคอยมองแป้มพิมพ์และ
             ใช้นิ้วจิ้มทีละตัว คุณอาจระบุเป้าหมายลงไปว่า จะต้องพิมพ์สัมผัสได้ในความเร็ว 30 คำต่อนาที 
             ภายในเวลา 3 วัน โดยการฝึกพิมพ์นี้คุณสามารถดาวน์โหลดวิธีฝึกออนไลน์ที่ไม่เสียค่าใช้จ่ายมา ใช้ในการฝึกได้ 
             หลังจากการฝึกในแต่ละครั้งให้คุณให้คะแนนตัวเองโดยเปรียบเทียบจากการฝึก ในครั้งที่ผ่านมาว่า
             คุณมีความก้าวหน้ามากเพียงใด เมื่อครบตามกำหนด 3 วัน มาดูกันว่าคุณสามารถทำได้ตามเป้าหมายหรือไม่ 
            จากนั้นทำการกำหนดเป้าหมายต่อ ๆ ไปที่คุณต้องการพัฒนาทักษะคอมพิวเตอร์เพิ่มเติม

            
 2. ทำได้แล้ว ทำให้เก่งขึ้นได้อีก

                        เมื่อคุณเข้าใจพื้นฐานจนสามารถใช้โปรแกรมคอมพิวเตอร์ได้แล้ว คุณอาจลองฝึกทักษะคอมพิวเตอร์
            ในระดับที่มีความซับซ้อนและยากขึ้นกว่าเดิม เช่น หากคุณใข้ Microsoft Excel สำหรับการใส่สูตรคำนวณ
            พื้นฐานอย่างง่ายๆ เช่น บวก ลบ คูณ หาร คุณอาจลองผูกสูตรคำนวณต่างๆเพิ่มเติม อาทิ
             การคิดหาเปอร์เซนต์กำไรขาดทุน หรือ การเทียบสัดส่วนของสองข้อมูล เป็นต้น หากคุณเข้าใจในทักษะเบื้องต้นเหล่านี้ 
             เป็นอย่างดีแล้วล่ะก็ คุณจะสามารถนำไปต่อยอดสู่ทักษะคอมพิวเตอร์ในขั้นที่สูงขึ้นต่อไปได้ง่ายยิ่งขึ้น

             
3. อยากเก่งต้องฝึกฝน 

                         วิธีเดียวที่จะทำให้คุณเก่งสำหรับทักษะคอมพิวเตอร์คือ ใช้งานให้บ่อยๆ ยิ่งคุณศึกษาด้วยตัวเอง
            ถึงฟังก์ชั่นต่างๆ ทั้งฮาร์ดแวร์และซอฟท์แวร์ รวมถึงวิธีการใช้งานคุณสมบัติต่าง ๆ ด้วย ความสงสัยจะทำให้คุณอยากรู้ 
            และขวนขวายด้วยตัวเอง ไม่มีใครที่เก่งได้ภายในวันเดียว ฝึกฝนการใช้งานต่างๆเพื่อเพิ่มพูนพัฒนาทักษะคอมพิวเตอร์
            ให้เก่งยิ่งขึ้น

            
4. อย่ากลัวการลองทำอะไรใหม่ๆ 

                         ผู้ที่เริ่มใช้คอมพิวเตอร์ ส่วนใหญ่มักจะกลัวว่าหากลองทำสิ่งใหม่ๆอาจเผลอไปลบไฟล์หรือ
            โปรแกรมสำคัญทิ้งไปจนอาจทำให้คอมพิวเตอร์พังเสียหายได้ หากคุณกลัวว่าจะเกิดความเสียหาย 
            ให้ทำการสำรองไฟล์ข้อมูลไว้ก่อนจะทำการแก้ไขก่อน ที่คุณจะทำอะไรกับคอมพิวเตอร์ของคุณ 
            วิธีนี้ทำให้คุณยังมีไฟล์สำรองใช้อยู่ในกรณีที่เกิดความผิดพลาดดังกล่าว นอกจากนี้คุณยังสามารถหาข้อมูล
            บนอินเตอร์เน็ตเพิ่มเติมได้ หรืออาจลองถามผู้ที่ใช้โปรแกรมดังกล่าวในกรณีที่คุณมีข้อสงสัยเพิ่มเติมต่างๆ
