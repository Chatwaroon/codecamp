declaration ความหมาย ตัวอย่าง
property คุณสมบัติ width, height, position

2.property ที่สอนใน precamp มีกี่กลุ่มอะไรบ้าง

ตอบ 5กลุ่ม

property ตัวอย่าง
Property for content color , size , font , text, list-style-type, Height, Width, background
property for border border, table
property for padding padding (top right buttom left)
property for margin margin (top right buttom left)
property for element display, Height, Width, Position, Float, Align, Flex, Grid

3.value unit มีกี่ประเภทอะไรบ้างแต่ละ ประเภทคืออะไร

ตอบ 2 ประเภท Absolute lenghts,relative lenghts

Absolute lenghts ความหมาย
cm เซนติเมตร
mm มิลลิเมตร
in นิ้ว
px พิกเซล 1/96 นิ้ว
pt พอยท์ 1/72 นิ้ว
pc พิคาล 1/12 นิ้ว

relative lenghts ความหมาย
em เท่าขนาดฟอนต์ของ element
ch ความกว้างของตัวหนังสือใน parent element
rem เท่าขนาดฟอนต์ของ root element
vh เปอร์เซ็นต์ความสูงหน้าจอ
vw เปอร์เซ็นต์ความกว้างหน้าจอ
vmin เปอร์เซ็นต์ของ view point ต่ำสุด
vmax เปอร์เซ็นต์ของ view point สูงสุด
% เปอร์เซ็นต์ของ parent element
