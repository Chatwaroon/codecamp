1. value ใน display property มีอะไรบ้าง

Display ความหมาย
inline แท็ก span อยู่บรรทัดเดียวกัน ไม่สามารถปรับแต่ง ความกว้าง ความสูงได้
block แท็ก p อยู่คนละบรรทัด
contents ทําให้ elementนัน้กลายเป็น contents ไม่สามารถปรับ margin, padding, border, background ได้
ﬂex การจัดรูปแบบ ﬂex
grid การจัดรูปแบบ grid
run-in block or inline ขี้นอยู่กับ เนื้อหา

2.grid และ flex แตกต่างกันอย่างไร

display property ความหมาย
grid ใช้เพื่อจัดการ layout 2D สามารถจัดของไว้ตรงไหนก็ได้ใน grid layout
flex ใช้ flex เมื่อ layout มีลักษณะการวางของต่างๆเป็นแถวเดียว ไม่แนวตั้งก็นอน (1D)

3.justify and align แตกต่างกันอย่างไร

display property ความหมาย
justify จัดการแนวนอน และ ใช้เหมือนกันใน align-content (ﬂex)
align ใช้กับ ﬂex จัดการแนวตั้ง
