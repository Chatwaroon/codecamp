1. position คืออะไร และ อะไรบ้าง

position คือ คุณสมบัติระบุชนิดของวิธีการวางตำแหน่งที่ใช้สำหรับองค์ประกอบ

มี position ดังนี้

position ความหมาย
static เป็นค่าพื้นฐานของทุกบล็อกโดนจะเลื่อนตามหน้าเว็บปกติ
relative การเลื่อนออกจากตําแหน่งเดิม ตามที่เรากําหนดการดัน block จากทาง ซ้าย ขวา บน ล่าง
ﬁxed กําหนดให้ block นัน้ยึดอยู่กับที่บนหน้าเว็บ
alsolute การเลื่อนออกจากมุมซ้ายบนสุดของหน้าจะ ตามการกําหนดการดัน block จากทาง ซ็าย ขวา บน ล่าง
sticky การรวม relative กับ ﬁxed ไว้ด้วยกัน โดยการกําหนด relative เพิ่มว่าจะให้ไป ﬁx ที่ top หรอื bottom ทําให้เราเห็น block นั้นเลื่อนและถูกตรงีไว้เมื่อเรา เลื่อน scroll เมาส์

2.z-index คืออะไรใช้อย่างไร

ตอบ เป็นการจัดการ element ในรูปแบบชัน้ layer บนหน้าจอโดยค่าน้อยจะอยู่ด้านหลัง ส่วนค่ามากจะอยู่ด้านหน้า ค่าเร่มิต้นของทุก element คือ z-index = 0
